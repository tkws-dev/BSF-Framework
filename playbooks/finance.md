## Revenue Model Options

### Option A: Commission (Marketplace)
```
We take X% of each transaction
Pros:
  - Aligns with value delivered
  - Scales with volume
Cons:
  - Need high volume to be meaningful
  - Disintermediation risk
```

### Option B: Flat Fee (Per Use)
```
Fixed price per transaction
Pros:
  - Predictable revenue
  - Simple to explain
Cons:
  - May discourage high-value transactions
```

### Option C: Subscription (Recurring)
```
฿___/month for unlimited access
Pros:
  - Recurring revenue
  - Predictable cash flow
Cons:
  - Need strong value proposition
  - Higher churn risk if value not clear
```

### Option D: Freemium
```
Free tier + paid upgrade
Pros:
  - Low barrier to entry
  - Viral growth potential
Cons:
  - Free → paid conversion typically <5%
  - Free users still cost money
```
