## Legal Research Framework

### 1. Business Registration

```
[ ] จดทะเบียนบริษัท? รูปแบบไหน?
    - บุคคลธรรมดา (ส่วนตัว)
    - ห้างหุ้นส่วน
    - บริษัทจำกัด
    - Recommendation:

[ ] ทุนจดทะเบียนขั้นต่ำ? 
    - สำหรับธุรกิจนี้: ________
    - สำหรับต่างชาติ (ถ้ามี): ________

[ ] ภาษีที่เกี่ยวข้อง:
    - ภาษีเงินได้นิติบุคคล: ___%
    - VAT: 7% (ถ้ารายได้ > 1.8M/ปี)
    - หัก ณ ที่จ่าย: ___%
```

### 3. PDPA / Data Privacy

```
[ ] เก็บข้อมูลอะไรบ้าง?
    - Personal data: ชื่อ, เบอร์, อีเมล, ที่อยู่
    - Sensitive data: ________
    - รูปภาพ: [YES/NO]

[ ] Consent:
    - ขอ consent ตอนไหน? (registration / transaction)
    - Opt-in หรือ Opt-out?

[ ] Data retention:
    - เก็บนานแค่ไหน?
    - ลบยังไง?

[ ] Data Processing Agreement (DPA):
    - ระหว่าง platform กับ service provider? [YES/NO]

[ ] Cross-border data transfer? [YES/NO]

[ ] ต้องมี DPO (Data Protection Officer)? [YES/NO]
    - เกณฑ์: เก็บ sensitive data จำนวนมาก
```

### 4. Terms of Service & Contracts

```
[ ] ต้องมีเอกสารอะไร?
    - Terms of Service (สำหรับลูกค้า): [YES/NO]
    - Service Agreement (สำหรับ provider): [YES/NO]
    - Privacy Policy: [YES/NO — mandatory ถ้าเก็บข้อมูล]
    - Cookie Policy: [YES/NO]

[ ] Contract template สำหรับ:
    - ระหว่าง platform ↔ customer
    - ระหว่าง platform ↔ service provider
    - ระหว่าง customer ↔ service provider (disclaimer)
```

### 5. Liability & Risk

```
[ ] ความเสี่ยงทางกฎหมาย:
    1. ________ (probability: สูง/กลาง/ต่ำ)
       Mitigation: ________
    2. ________ 
       Mitigation: ________

[ ] Insurance:
    - Platform ต้องมีประกันอะไร?
    - Provider ต้องมีประกันอะไร?

[ ] Dispute resolution:
    - กระบวนการรับเรื่องร้องเรียน
    - Refund policy
    - Arbitration clause
```

### 6. E-Commerce / Digital Platform Laws

```
[ ] พ.ร.บ. ว่าด้วยธุรกรรมทางอิเล็กทรอนิกส์
    - Electronic signature: [YES/NO]
    - E-contract validity

[ ] ประกาศ คคบ. (คณะกรรมการคุ้มครองบริโภค):
    - สินค้า/บริการที่ควบคุม? [YES/NO]

[ ] DBD (กรมพัฒนาธุรกิจการค้า):
    - จดทะเบียนพาณิชย์อิเล็กทรอนิกส์? [YES/NO]
```
